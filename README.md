# Enterprise Asset Management (EAM) Dashboard | ServiceNow

A custom **Enterprise Asset Management (EAM)** application built on the **ServiceNow Platform** to simplify and centralize enterprise asset operations.

This project demonstrates how ServiceNow can be customized to provide an intuitive dashboard for managing assets, employees, departments, vendors, requests, allocations, returns, and reports through a modern user interface.

---

## 📖 About the Project

Managing enterprise assets across multiple departments can become challenging when information is distributed across different systems. This project provides a centralized dashboard that improves visibility, organization, and efficiency for asset management processes.

The application was developed as part of my ServiceNow learning journey to gain hands-on experience with enterprise application development and UI customization.
